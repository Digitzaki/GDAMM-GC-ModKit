#### **============= EXTRACT ISO =============**



**(You may ignore this part if already done)**



1\) In dolphin right click the game --> Properties

~~------------------------------------------------------------------~~

2\) Go to file system and right click on the disc --> Extract entire disc

~~------------------------------------------------------------------~~

3\) Location --> I recommend the Dolphin Game directory for step 4   *(Same place you load your Roms)*





**4) In Config go to paths and check "Search Subfolders", You can now run the extracted game like normal!!**





##### ============ GETTING STARTED ==============



(In the AI files are found in **files/ShellData** ...Data? For a Gas Station?)



1\) Open the AI Editor Tool.

~~------------------------------------------------------------------~~

2\) Hit Open Config and pick any of the following:



**"ai\_easy.txt"
"ai\_medium.txt"
"ai\_hard.txt"**



(Custom Adventure Modes will come later... maybe.)



~~------------------------------------------------------------------~~



* MIN\_DIFFICULTY = Minimum Difficulty AI can reach. (ex. "0.3")



* MAX\_DIFFICULTY = Maximum Difficulty AI can reach. (ex. "0.7")



* MIN\_HEALTH DELTA = Always a subtraction value (-), how much the range above decreases (ex. "**-**0.15" = if ai is at 0.5 **--> 0.35**)



* MAX\_HEALTH\_DELTA = Always an addition value (+), how much the range above decreases (ex. "**+**0.15" = if ai is at 0.5 **--> 0.65**)



* DIFFICULTY\_DELTA **(Survival Only)**:  How much the MIN and MAX difficulty ranges will increase per round. (ex. "**0.1"** **=(round 1)=** 0.3 - 0.5 **=(round 2)=** 0.4 - 0.6)



* UPDATE\_TIME: How frequent MIN and MAX Delta shift the range. (ex: "3" = 3 seconds per delta update)
