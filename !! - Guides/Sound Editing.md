#### 

#### **============= EXTRACT ISO =============**



**(You may ignore this part if already done)**



1\) In dolphin right click the game --> Properties

~~------------------------------------------------------------------~~

2\) Go to file system and right click on the disc --> Extract entire disc

~~------------------------------------------------------------------~~

3\) Location --> I recommend the Dolphin Game directory for step 4   *(Same place you load your Roms)*





**4) In Config go to paths and check "Search Subfolders", You can now run the extracted game like normal!!**





##### ============ Extract Sounds ==============



(In the Extracted Disc Sounds are found in files/SoundData)  



1\) Open a Godzilla\_sound **.samp** file and its corresponding **.ZIP**



&#x20;       **- I recommend making a copy somewhere else with the .samp and extracted zip**



~~------------------------------------------------------------------~~

2\) Open Godzilla Sound Manager in Tools





##### **============== SET UP ================**



\- Drag or open the .sdir from inside the extracted zip into GSM

\- Drag or open the .samp from inside the extracted zip into GSM



\- In the bottom left you will see every sound appear with a preview button!



&#x20;      - This ensures you find the ones you want, its one big ugly library... ew.



~~------------------------------------------------------------------~~



\- Look around and preview these crusty noises, and check the ones you want to use.



&#x20;               - Every character has their sounds next to each other

&#x20;

\- While Extract .dsp gives you the raw file. **I recommend Extract .WAV for editing.**





##### **=============== EDITING GUIDELINES ==============**



**(I use Audacity, but anything that can export a new .WAV is ok)**



Put your extracted sound into the editor and **MUTE IT!!!!**

~~------------------------------------------------------------------~~

Sounds cannot be longer, but they *can* be shorter. longer sounds get cut.



Muting the original means you know exactly how long it needs to be!





##### **================ Rebuild ======================**



**1)** Once your done with the sound Extract it and REPLACE the extracted wav with the new one.

~~------------------------------------------------------------------~~

**2)** Ensure these extracted .wav(s) are in the same place as the sound files

~~------------------------------------------------------------------~~

**3)** Set up GSM again if needed, and hit rebuild. 



It'll detect the wavs, inject them in, and auto encode them back into the correct GC format!





##### **============== LOAD INGAME ===================**



Option 1: Put the file back in the extracted disc from above



Option 2: Use Riivolution, But it requires some setup... My mod Godzilla: DAM - Domination! Comes with loading custom files



Either of these options and it'll load in-game!
